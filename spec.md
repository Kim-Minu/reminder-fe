# PRD — Apple Reminders Web Version (Frontend)

## 1. 프로젝트 개요

### 배경
Apple Reminders는 iOS/macOS 생태계에서만 동작하는 할 일 관리 앱이다. 플랫폼에 종속되지 않고 브라우저에서 동일한 경험을 제공하는 웹 버전을 개발한다.

### 목표
- Apple Reminders의 핵심 UX를 웹에서 재현한다.
- 빠르고 직관적인 할 일 관리 경험을 제공한다.
- 모바일/데스크탑 반응형으로 어느 기기에서도 사용 가능하게 한다.

### 범위 (MVP)
- 리마인더 목록(List) 및 리마인더(Reminder) CRUD
- 스마트 목록 (오늘, 예정됨, 전체, 완료됨)
- 마감일/시간, 우선순위, 메모, 플래그, 반복 설정
- 검색

## 3. 핵심 기능

### 3.1 리마인더 목록 (List) 관리
- 목록 생성 / 이름 수정 / 삭제
- 목록별 색상 설정 (Apple 기본 팔레트 16색)
- 목록별 리마인더 개수 배지 표시
- 목록 순서 드래그 정렬

### 3.2 리마인더 (Reminder) 관리
- 완료 토글 (체크박스)
- 제목 인라인 편집
- 마감일 / 마감 시간 설정
- 우선순위: 없음 / 낮음(`!`) / 중간(`!!`) / 높음(`!!!`)
- 메모(notes) 멀티라인 입력
- 플래그(깃발 아이콘) 토글
- 반복 설정: 없음 / 매일 / 매주 / 매월 / 매년
- 하위 리마인더(sub-reminder) — 선택 기능

### 3.3 스마트 목록
| 스마트 목록 | 조건 |
|------------|------|
| 오늘 | 마감일이 오늘이거나 기한 초과 (미완료) |
| 예정됨 | 마감일이 내일 이후 (미완료) |
| 전체 | 모든 미완료 리마인더 |
| 완료됨 | 완료 처리된 리마인더 |
| 플래그됨 | 플래그가 설정된 미완료 리마인더 |

### 3.4 검색
- 전체 리마인더 제목 / 메모 대상 실시간 검색
- 검색 결과에서 해당 목록 정보 표시

## 5. UI/UX 디자인 — Apple Reminders 동일 구현

> Apple Reminders macOS/iOS 앱의 시각적 스타일과 인터랙션을 최대한 동일하게 재현한다.

### 5.1 전체 레이아웃 (macOS Reminders 기준)

```
┌──────────────────────────────────────────────────────────────┐
│  사이드바 (260px)       │  리마인더 목록          │ 상세패널  │
│ ─────────────────────  │ ───────────────────── │ ────────  │
│  🔍 검색               │  [목록명, 굵게, 대형]   │  제목     │
│                        │                       │  메모     │
│  스마트 목록            │  새 리마인더 추가...    │  ───────  │
│  ☀️  오늘          3   │  ─────────────────    │  마감일   │
│  📅 예정됨         8   │  ○  리마인더 A         │  마감시간 │
│  📋 전체          21   │     오늘 · 메모 미리보기│  우선순위 │
│  ✓  완료됨             │  ○  리마인더 B   🚩    │  플래그   │
│  🚩 플래그됨       2   │     내일 오전 9:00     │  반복     │
│                        │  ○  리마인더 C   !!!  │  목록     │
│  나의 목록             │                       │           │
│  🔴 집           12   │  ▾ 완료됨              │           │
│  🔵 직장           3   │  ✓  완료된 항목        │           │
│  🟢 쇼핑           0   │                       │           │
│  ＋ 목록 추가          │                       │           │
└──────────────────────────────────────────────────────────────┘
```

### 5.2 색상 & 테마
| 요소 | 값 |
|------|----|
| 배경 (사이드바) | `#F2F2F7` (macOS 사이드바 회색) |
| 배경 (메인) | `#FFFFFF` |
| 텍스트 기본 | `#1C1C1E` |
| 텍스트 보조 | `#8E8E93` |
| 완료된 항목 텍스트 | `#8E8E93` + 취소선 |
| 강조색 | 목록별 색상 (기본 `#007AFF` Apple Blue) |
| 구분선 | `#E5E5EA` |
| 다크모드 | 지원 (`prefers-color-scheme`) |

### 5.3 사이드바
- 섹션 구분: **스마트 목록** / **나의 목록**
- 스마트 목록 아이콘: SF Symbol 스타일 컬러 아이콘 (Lucide 대체)
  - 오늘: 달력+숫자 아이콘, 파란색 원형 배경
  - 예정됨: 달력 아이콘, 빨간색 원형 배경
  - 전체: 목록 아이콘, 회색 원형 배경
  - 완료됨: 체크 아이콘, 회색 원형 배경
  - 플래그됨: 깃발 아이콘, 주황색 원형 배경
- 사용자 목록: 색상 원형 아이콘 + 목록명 + 미완료 개수 배지
- 선택된 항목: 파란색 배경 하이라이트, 흰색 텍스트
- `+ 목록 추가` 버튼: 하단 고정

### 5.4 리마인더 목록 영역
- **목록 타이틀**: 목록 색상으로 굵고 크게 (32px), 미완료 개수 표시
- **새 리마인더 추가**: 목록 최상단, `+ 새 리마인더` 플레이스홀더 — 클릭 시 인라인 입력 즉시 활성화
- **리마인더 행**:
  - 왼쪽: 원형 체크박스 (목록 색상), 클릭 시 체크 애니메이션 후 완료 처리
  - 제목 + 메모 미리보기 (1줄 말줄임)
  - 마감일/시간 (기한 초과 시 빨간색)
  - 우선순위 느낌표 (`!` / `!!` / `!!!`, 빨간색)
  - 플래그 아이콘 (주황색)
- **완료된 항목**: 별도 섹션 `▾ 완료됨` 으로 접기/펼치기
- **행 호버**: 배경 회색으로 미세 변경, 우측에 `...` 컨텍스트 메뉴 버튼 노출

### 5.5 상세 편집 패널 (우측 슬라이드)
- 리마인더 클릭 시 우측에서 슬라이드인 (350px 고정)
- 필드 구성 (Apple Reminders 순서 동일):
  1. 제목 (대형 텍스트 입력)
  2. 메모 (멀티라인 텍스트)
  3. 구분선
  4. 마감일 (날짜 피커)
  5. 마감 시간 (시간 피커)
  6. 반복 (드롭다운)
  7. 구분선
  8. 우선순위 (없음 / ! / !! / !!!)
  9. 목록 (드롭다운, 목록 이동 가능)
  10. 플래그 토글
- 각 행: 아이콘 + 레이블 + 값 (iOS Settings 스타일)

### 5.6 인터랙션 상세
| 인터랙션 | 동작 |
|---------|------|
| 체크박스 클릭 | 체크 애니메이션 → 0.3s 후 목록에서 페이드아웃 → 완료 섹션 이동 |
| 제목 클릭 | 인라인 편집 모드 활성화 (별도 페이지 이동 없음) |
| Enter 입력 | 현재 리마인더 저장 + 다음 리마인더 입력 즉시 시작 |
| Escape | 편집 취소 |
| 드래그 | 목록 내 리마인더 순서 변경 |
| 우클릭 | 컨텍스트 메뉴 (삭제, 플래그, 마감일 설정) |
| 목록 더블클릭 | 목록 이름 인라인 편집 |

### 5.7 목록 생성 모달
- Apple 스타일 시트 모달 (하단에서 올라오는 바텀시트 — 모바일 / 데스크탑 센터 모달)
- 목록 아이콘 미리보기 (선택 색상 원형)
- 이름 입력 + 색상 팔레트 16색 선택
- `취소` / `완료` 버튼

### 5.8 주요 컴포넌트
| 컴포넌트 | 설명 |
|---------|------|
| `AppLayout` | 3-panel 레이아웃 (사이드바 / 목록 / 상세) |
| `Sidebar` | 스마트 목록 + 나의 목록 내비게이션 |
| `SmartListIcon` | 컬러 원형 배경 + 아이콘 |
| `ReminderListView` | 선택된 목록의 리마인더 목록 |
| `ReminderRow` | 체크박스, 제목, 메모 미리보기, 배지 |
| `ReminderDetailPanel` | 우측 슬라이드 상세 편집 패널 |
| `InlineReminderInput` | 인라인 새 리마인더 입력 |
| `ColorPicker` | 16색 팔레트 |
| `ListFormModal` | 목록 생성/수정 모달 |
| `SearchOverlay` | 전체화면 검색 오버레이 |
| `ContextMenu` | 우클릭 메뉴 |

### 5.9 반응형 동작
| 화면폭 | 동작 |
|--------|------|
| ≥ 1024px | 3-panel (사이드바 + 목록 + 상세패널) |
| 768px ~ 1023px | 2-panel (사이드바 + 목록, 상세패널은 오버레이) |
| < 768px | 1-panel (사이드바 → 목록 → 상세 순차 이동) |

| 항목 | 요건 |
